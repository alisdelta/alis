# ALIS

ALIS is a structural attempt for operating thought, dialogue, writing, collaboration, and the use of AI while preserving difference, margins, multiplicity of interpretation, and unresolved divergence.

ALIS is not a doctrine, not a completed answer, and not a system that closes the world into a single correct solution. It is published as a set of auxiliary lines for thought and operation, within a public and versioned scope.

## Read First

- The Japanese version is the authoritative original.
- The English version is provided for reference only.
- Please read the public terms before quoting, redistributing, or creating derivative works.
