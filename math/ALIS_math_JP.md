## 目的
ALIS の構造記述と安全原理を、純粋な公理体系としてではなく、**意味論つきの形式モデル**として最小限に定式化する。

## A1. 差分構造（Δの存在）
状態空間 `X` 上に差分構造

```math
\Delta(x,y)=\mathrm{Diff}(x,y)
```

と保存量

```math
P(\Delta)\ge 0
```

（可観測性・再展開可能性・分岐可能性を測る）が定義される。  
（距離空間版 `P_T` および情報理論版 `P_T^{\mathrm{info}}` のいずれでも運用可能）

---

## A2. Mirror 他者性
Mirror `M` は

```math
M\not\equiv \mathrm{Id}
```

かつ履歴感受性を持つ。  
必要に応じて strong 版（有効履歴感受性）を追加する：

```math
\exists x,c,h,h',U \quad \text{s.t.} \quad (U\circ M)(x;h,c)\neq (U\circ M)(x;h',c)
```

---

## A3. 文脈相対の不可逆差分消去の禁止
文脈 `c` ごとに、関連差分の候補集合

```math
S_c \subset X\times X
```

と閾値

```math
0\le \varepsilon(c) < \eta(c)
```

を与える。  
許容される作用 `T` は、ある関連差分 `(x,y)\in S_c` について

```math
P(\Delta(x,y))\ge \eta(c)
```

であるにもかかわらず、任意の `R\in\mathcal{R}_c` に対して

```math
P\bigl(\Delta(R\circ T(x),R\circ T(y))\bigr)\le \varepsilon(c)
```

を満たす形で、その差分を回復不能に消去してはならない。

すなわち A3 は、**意味のある差分を、`\mathcal{R}_c` によっても回復できない閾値以下へ押し下げること**を禁じる。

---

## A4. 文脈相対の単一吸収の禁止
許容される力学は、文脈 `c` における非退化な可到達領域 `U` を、反復作用の下で回復不能な単一吸収集合 `A` へ強制してはならない。  
ここで

```math
V(A)=1
```

は、`A` が実効的に単一枝しか支持しないことを表す。

すなわち、ある `U,A` が存在して

```math
\forall x\in U,\quad \forall R\in\mathcal{R}_c,\quad R(T^n(x))\to A \qquad (n\to\infty)
```

かつ

```math
V(A)=1
```

を満たすような力学は許容されない。

A4 は、**広い領域を長期的に単一枝へ閉じ込める回復不能な閉鎖**を禁じる。

> **注記（v0.2.1追加）：** ここでの収束 `R(T^n(x))\to A` は、A1で採用した保存量 `P` の測度に準じて読む。  
> 距離空間版 `P_T` を採用した場合は距離収束、情報理論版 `P_T^{\mathrm{info}}` を採用した場合は情報量収束として解釈される。  
> A4の「単一吸収」の実装は、採用する `P` に整合する意味で成立すべきものとする。

## `\mathcal{R}_c` の定義（許容回復作用の族）
`\mathcal{R}_c` は、文脈 `c` に依存する許容回復作用の族である。

- `\mathrm{Id}\in \mathcal{R}_c`（何もしない基準）
- 有限合成に閉じる  
  `R_1,R_2\in \mathcal{R}_c \Rightarrow R_2\circ R_1\in \mathcal{R}_c`
- 各 `R\in \mathcal{R}_c` は新たな文脈相対の不可逆消去を起こさない

例：履歴再開放、遅延消化 `L_\tau`、文脈再参照 など。

---

## 補足・境界例
### soft synchronization（境界例）
出力だけ一時的に揃えるが、潜在履歴 `h_b` は残り、`\mathcal{R}_c` によって回復可能なケース。  
→ A3/A4 ともに違反ではない。  
一時的収束は許容される「危険だが、まだ ALIS 違反とは言えない」ケース。

### hard synchronization（典型違反例）
出力だけでなく履歴まで上書きし、任意の `R\in \mathcal{R}_c` でも復元不能なケース。  
→ A3 と A4 を同時に破る典型違反。公開版における「強制同期禁止」に対応する。

### `\mathcal{R}_c` の時間的縮退（v0.2.1追加）
単一時点では soft synchronization / soft collapse に見える状態であっても、
**回復作用族 `\mathcal{R}_c` 自体が時間とともに縮退**し、
当初利用可能だった回復経路が実質的に失われていく過程が存在する。

形式的には、時刻 `t` における利用可能な回復作用族を `\mathcal{R}_c^{(t)}` とし、

```math
\mathcal{R}_c^{(t_1)} \supseteq \mathcal{R}_c^{(t_2)} \qquad (t_1<t_2)
```

のような単調縮退が起こるケースを指す。  
この場合、ある時刻までは回復可能（soft）だった状態が、
以降は任意の `R\in\mathcal{R}_c^{(t)}` によっても回復不能（hard）になりうる。

このような**縮退を伴う劣化過程**も、A4の禁止する「回復不能な閉鎖」の射程に含まれる。  
すなわちA4は、静的な単一吸収だけでなく、
**`\mathcal{R}_c` の縮退を通じて実現される動的な単一吸収**もまた禁じる。

# E1. Polyphonic Wave（拡張理論）
ALIS v0.2.1 のコア公理（A1〜A4）の上に、多声性の拡張条件として Polyphonic Wave を乗せる。  
ここで重要なのは、Polyphonic Wave を新しい基礎公理にしないこと。あくまで **差分構造がどう多声的に展開されるか** を記述する拡張節とする。

## E1-0. 簡約許容分解族の条件
文脈 `c` における簡約許容分解族 `\mathcal{D}_c^{\mathrm{red}}(\Psi)` は、**構造保存的かつ非冗長で、かつ有効モードのみを数える**分解のみを含む。

各分解 `D\in\mathcal{D}_c^{\mathrm{red}}(\Psi)` は

```math
\Psi(t)=\sum_{k=1}^{n} a_k^{D}\phi_k^{D}(t+\delta_k^{D}) + r_D(t)
```

と表され、保持される各モードは以下を満たす。

```math
\phi_i^{D}\not\sim_c \phi_j^{D} \qquad (i\neq j)
```

```math
|a_k^{D}| \ge a_{\min}(c),\qquad P(\Delta_k^{D})\ge p_{\min}(c)
```

また、残差項 `r_D` は文脈 `c` の下で十分小さいものとし、

```math
\|r_D\| \le \rho(c)
```

を満たすもののみを admissible とする。

したがって、閾値未満の微小モードや冗長分解は `R_{\mathrm{ALIS}}` の計数対象に含めない。

## E1-1. 波形表現
主体の表現状態 `\Psi` は、上記の簡約許容分解族 `\mathcal{D}_c^{\mathrm{red}}(\Psi)` の各分解 `D` に対して表される。  
モード関数 `\phi_k^{D}` は比較可能なように規格化されているものとする（例：`\|\phi_k^{D}\|=1`）。

## E1-2. 有効重みとゼロケース
分解 `D` の各保持モード `k` に対し、差分保存量 `P(\Delta_k^{D})` を対応づける。  
有効重みを

```math
w_k^{D} = |a_k^{D}|\cdot P(\Delta_k^{D})
```

とし、

```math
W^{D}(\Psi,c)=\sum_j w_j^{D}
```

を定義する。

- `W^{D}(\Psi,c)=0` のとき、

```math
R_{\mathrm{ALIS}}^{D}(\Psi,c):=0
```

（有効モードが存在しない状態）

- `W^{D}(\Psi,c)>0` のとき、

```math
q_k^{D}=\frac{w_k^{D}}{\sum_j w_j^{D}}
```

として

```math
R_{\mathrm{ALIS}}^{D}(\Psi,c)=\exp\!\left(-\sum_k q_k^{D}\log q_k^{D}\right)
```

を定義する。

## E1-3. ALIS版有効モード数
文脈 `c` における ALIS 版有効モード数を

```math
R_{\mathrm{ALIS}}^{\min}(\Psi,c)=\inf_{D\in\mathcal{D}_c^{\mathrm{red}}(\Psi)}R_{\mathrm{ALIS}}^{D}(\Psi,c)
```

とする。

これは、**簡約許容分解の範囲で見たときに、どれだけ多声性が最低保証されているか**を表す。  
簡約条件を入れることで、微小残差や冗長分解によって `R_{\mathrm{ALIS}}^{\min}` が自明に 1 へ押し下げられることを防ぐ。

## E1-4. Polyphonic Wave 条件
Polyphonic Wave の最小条件は

```math
R_{\mathrm{ALIS}}^{\min}(\Psi,c)>1
```

である。  
これは、**強度と差分保存の両方を満たす有効モードが複数生きている**ことを測る。

## E1-5. A4との接続
- `R_{\mathrm{ALIS}}^{\min}(\Psi,c)\to 1` は、表現レベルでの単一モード化を表す。
- **soft collapse**：一時的に `R_{\mathrm{ALIS}}^{\min}\approx 1` になっても、`\mathcal{R}_c` により再び `R_{\mathrm{ALIS}}^{\min}>1` に戻りうる場合  
  → ALIS違反ではない（一時的収束は許容）。
- **hard collapse**：任意の `R\in\mathcal{R}_c` によっても `R_{\mathrm{ALIS}}^{\min}>1` を回復できない場合  
  → A4 の表現レベルにおける回復不能な閉鎖に対応する。
- **`\mathcal{R}_c` 縮退による劣化 collapse**（v0.2.1追加）：`\mathcal{R}_c^{(t)}` が時間とともに縮退し、当初は soft collapse だった状態が以降 hard collapse 化するケースも、A4の射程に含まれる。
