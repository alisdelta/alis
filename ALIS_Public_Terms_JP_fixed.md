## 前文

本方針は、GitHubその他の公開媒体において公開される ALIS 関連文書、定義、補注、図式、実践例、要約および付随情報（以下「本公開物」）について、その利用条件と公開上の基本方針を定めるものです。

ALIS は、単一の正解へと世界を閉じるための体系ではなく、差分、余白、複数解釈性、未回収のズレを保持したまま、人間の思考、対話、記述、協働およびAI利用を壊れにくく運用するための構造的試みです。

したがって、本公開物は、誰かを従わせるための教義としてではなく、公開可能な範囲で共有される設計上の補助線として扱われます。本公開物を閲覧、引用、参照、共有または派生利用する者は、本方針の趣旨を理解したうえで利用するものとします。

## 第2条（定義）

本方針において、以下の用語は次の意味で使用されます。

1. **「公開主体」** とは、ALIS の開発者であり、本公開物の公開・更新・撤回に関する最終判断を行う者をいいます。本方針の公開時点における公開主体の識別は、以下の公式チャネルによって行われます。
   - X アカウント：@alis_delta
   - GitHub リポジトリ：https://github.com/alisdelta/alis

   上記チャネルから公開されたもの以外は、本方針における「本公開物」に該当しません。
2. **「本公開物」** とは、公開主体が ALIS 名義のもとで公開する文章、定義、図表、補遺、公開ノート、固定版文書、X投稿、README、ライセンス補足、Q&Aその他の公開物をいいます。
3. **「派生物」** とは、利用者が本公開物を参照、引用、翻案、再構成、比較、図式化または応用して独自に作成した文書、解説、実践メモ、翻訳その他の成果物をいいます。
4. **「正本」** とは、公開主体が最優先の参照元として明示した固定版文書をいいます。公開媒体が複数存在する場合、GitHubリポジトリ上の固定版文書を正本とします。

1. ALIS は、原則として公開可能な構造層を中心に公開されます。

3. 神話的、象徴的、人格化的、個人固有的な高密度領域は、原則として公開対象に含めません。
4. 公開は一括全面公開ではなく、固定版、版管理、補注、段階的公開その他の方法により行われる場合があります。
5. 公開の目的は、崇拝対象や従属対象を作ることではなく、利用者が自らの現実へ戻りつつ思考・対話・運用の補助線を得られるようにすることにあります。

- 原典が ALIS 公開物に依拠していること。

- 公開主体の保証を伴わない解釈に「公式」「正統」「公認」等の表示を付さないこと。

### 3. 再配布

固定版の再配布、ミラー、引用集の作成は可能ですが、版番号、出典、改変有無、正本への参照先を明示することが望まれます。
